== Description ==

Wordpress plugin for reviewing a candidate registered through a ninja-forms form.
It will help you:

*    List registered candidates.
*    View candidate info.
*    Review a candidate.

Stay tuned as it will get updated possibly often with new features.

== Installation ==
= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `talentua-tallermasters-songwriter-reviewer.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard
